My Projects

TumbleWe

An original application that allows users to automatically operate their web applications on subdomain, which used: AWS, PostgreSQL, JavaScript, React / Redux, HTML5, Material UI.

It allows logged in user to:
  • Upload their build files(including js, css, and html file)
  • Create selectors, queried by css(e.t. bt.add) in a certain order
  • Simulate user’s operations on the subdomain

NomNomRide

An e-commerce app inspired by UberEats. It has a Python/Flask backend and React front end.

It allows any user to:

  • Search restaurants

  • Mange owned restaurants and menu items
  • Make orders in the shopping cart
  • Make reviews to non-owned restaurants
